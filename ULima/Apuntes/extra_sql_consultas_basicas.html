1. Estructura Básica de Selección

La sentencia SELECT es el eje fundamental para recuperar información de las tablas.

-- Seleccionar columnas específicas
SELECT nombre, apellido, salario 
FROM empleados
WHERE salario > 2000
ORDER BY apellido ASC;

2. Filtros Avanzados

  • LIKE: Búsqueda de patrones (% para varios caracteres, _ para uno solo).
  • IN: Compara un valor contra una lista de opciones.
  • BETWEEN: Filtra valores dentro de un rango inclusivo.
-- Uso de LIKE
SELECT * FROM productos WHERE nombre LIKE 'Apple%';

-- Uso de BETWEEN
SELECT * FROM ventas WHERE fecha_pedido BETWEEN '2023-01-01' AND '2023-12-31';

4. Lógica Condicional (CASE)

Permite implementar lógica de "if-else" directamente en la consulta SQL.

SELECT nombre, 
       salario,
       CASE 
           WHEN salario > 5000 THEN 'Alto'
           WHEN salario > 2000 THEN 'Medio'
           ELSE 'Bajo'
       END AS nivel_salarial
FROM empleados;
