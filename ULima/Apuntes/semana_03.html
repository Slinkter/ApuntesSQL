💡 Insight del Arquitecto: Diseo Fsico en 2026

PostgreSQL vs Oracle: El material original usaba sintaxis Oracle (VARCHAR2, NUMBER). En PostgreSQL moderno usamos:

  • VARCHAR(n) en lugar de VARCHAR2(n)
  • INTEGER o SERIAL en lugar de NUMBER
  • TIMESTAMP o DATE para fechas
  • BOOLEAN para valores lgicos
  • JSONB para documentos

2. Creacin de Tablas

Sintaxis PostgreSQL:

CREATE TABLE nombre_tabla (
    columna1 tipo_dato [RESTRICCIONES],
    columna2 tipo_dato [RESTRICCIONES],
    ...
    [RESTRICCIONES_DE_TABLA]
);

Ejemplo: Tabla CLIENTE

-- PostgreSQL
CREATE TABLE cliente (
    id_cliente SERIAL PRIMARY KEY,
    nombre VARCHAR(50) NOT NULL,
    apellido VARCHAR(50) NOT NULL,
    email VARCHAR(100) UNIQUE,
    telefono VARCHAR(20),
    fecha_registro DATE DEFAULT CURRENT_DATE,
    activo BOOLEAN DEFAULT TRUE
);

⚠️ Equivalencias Oracle → PostgreSQL:

  • NUMBER → INTEGER o NUMERIC
  • VARCHAR2 → VARCHAR
  • NUMBER(7) PRIMARY KEY → SERIAL PRIMARY KEY

Restricciones de Tabla

CREATE TABLE orden_pedido (
    numero_orden INTEGER,
    fecha_colocacion DATE,
    fecha_llenado DATE,
    numero_embarques INTEGER,
    id_cliente INTEGER,
    
    -- Restricciones de tabla
    PRIMARY KEY (numero_orden),
    FOREIGN KEY (id_cliente) REFERENCES cliente(id_cliente)
        ON DELETE CASCADE
        ON UPDATE CASCADE
);

4. Clave Fornea con Opciones

Opciones de FOREIGN KEY:

  • ON DELETE CASCADE: Elimina filas hijos automticamente
  • ON DELETE SET NULL: Pone NULL en la FK
  • ON DELETE RESTRICT: Impide eliminar si hay hijos
  • ON UPDATE CASCADE: Actualiza automticamente

5. Alteracin de Tablas

-- Agregar columna
ALTER TABLE cliente ADD COLUMN observaciones TEXT;

-- Modificar columna
ALTER TABLE cliente ALTER COLUMN telefono TYPE VARCHAR(30);

-- Agregar restriccin
ALTER TABLE cliente ADD CONSTRAINT email_check 
    CHECK (email LIKE '%@%.%');

-- Eliminar columna
ALTER TABLE cliente DROP COLUMN observaciones;

-- Eliminar restriccin
ALTER TABLE cliente DROP CONSTRAINT email_check;

6. Eliminacin de Tablas

-- Eliminar tabla (solo si no hay dependencias)
DROP TABLE nombre_tabla;

-- Eliminar con todas las restricciones de FK
DROP TABLE nombre_tabla CASCADE;

-- Eliminar solo si existe
DROP TABLE IF EXISTS nombre_tabla;
