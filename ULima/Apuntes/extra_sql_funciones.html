Conceptos Clave

  • COALESCE Manejo de valores nulos.
  • NULLIF Comparación y nulidad.
  • CAST Conversión de tipos.
  • Fecha/Hora Operaciones temporales.

1. Manejo de Nulos (COALESCE)

La función COALESCE retorna el primer valor no nulo de una lista.

-- Si el teléfono es NULL, muestra 'No disponible'
SELECT nombre, COALESCE(telefono, 'No disponible') AS contacto
FROM clientes;

2. Conversión de Datos (CAST)

En PostgreSQL, podemos usar el operador :: o la función CAST().

-- Convertir un texto a entero
SELECT '123'::INTEGER + 7;

-- Usando sintaxis ANSI
SELECT CAST('2023-05-13' AS DATE);

3. Funciones de Fecha y Hora

Función
Descripción
NOW()
Fecha y hora actual del servidor.
AGE(fecha)
Calcula el intervalo transcurrido hasta hoy.
EXTRACT(part FROM fecha)
Obtiene año, mes o día de una fecha.

4. Operadores de Comparación Avanzados

-- Operador IS DISTINCT FROM (maneja nulos como valores comparables)
SELECT * FROM tabla WHERE col1 IS DISTINCT FROM col2;
