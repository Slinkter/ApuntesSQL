2. La clßusula GROUP BY

GROUP BY: Agrupa filas con valores iguales para aplicar funciones de agregaciˇn.

Ejemplos:

-- Contar clientes por tipo
SELECT tipo, COUNT(*) AS cantidad
FROM cliente
GROUP BY tipo;

-- Ventas por producto
SELECT id_producto, SUM(cantidad) AS total_vendido
FROM detalle_pedido
GROUP BY id_producto;

-- Promedio de precio por categorÝa
SELECT categoria, AVG(precio) AS precio_promedio
FROM producto
GROUP BY categoria
ORDER BY precio_promedio DESC;

-- M˙ltiples columnas en GROUP BY
SELECT categoria, proveedor, COUNT(*) AS productos
FROM producto
GROUP BY categoria, proveedor;

Regla: Toda columna en el SELECT que NO es funciˇn de agregaciˇn DEBE estar en el GROUP BY.

3. La clßusula HAVING

HAVING: Filtra grupos despuÚs de GROUP BY (equivale a WHERE pero para grupos).

Ejemplos:

-- Productos con mßs de 5 unidades vendidas
SELECT id_producto, SUM(cantidad) AS total
FROM detalle_pedido
GROUP BY id_producto
HAVING SUM(cantidad) > 5;

-- Clientes con mßs de 3 pedidos
SELECT id_cliente, COUNT(*) AS num_pedidos
FROM orden_pedido
GROUP BY id_cliente
HAVING COUNT(*) > 3
ORDER BY num_pedidos DESC;

-- Diferencia WHERE vs HAVING
SELECT categoria, AVG(precio) AS promedio
FROM producto
WHERE activo = TRUE          -- filtra FILAS antes de agrupar
GROUP BY categoria
HAVING AVG(precio) > 100;    -- filtra GRUPOS despuÚs de agregar

6. Expresiones Condicionales (CASE)

CASE: Equivalente al IF-THEN-ELSE en SQL.

Ejemplos:

-- CASE simple
SELECT nombre,
    CASE tipo
        WHEN 'A' THEN 'Premium'
        WHEN 'B' THEN 'Estßndar'
        WHEN 'C' THEN 'Bßsico'
        ELSE 'Otro'
    END AS categoria
FROM cliente;

-- CASE con condiciones
SELECT producto,
    CASE 
        WHEN precio > 1000 THEN 'Caro'
        WHEN precio > 500 THEN 'Medio'
        ELSE 'Barato'
    END AS rango_precio
FROM producto;

-- CASE en agregaciˇn
SELECT 
    SUM(CASE WHEN estado = 'COMPLETADO' THEN 1 ELSE 0 END) AS completados,
    SUM(CASE WHEN estado = 'PENDIENTE' THEN 1 ELSE 0 END) AS pendientes
FROM orden_pedido;
