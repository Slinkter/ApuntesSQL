1. DML - Lenguaje de Manipulación de Datos

  • INSERT: Añade nuevas filas a una tabla
  • UPDATE: Modifica valores de filas existentes
  • DELETE: Elimina filas de una tabla
  • SELECT: Consulta y recupera datos

2. INSERT - Añadir Datos

-- Insertar fila específica
INSERT INTO empleados (id, nombre, departamento, salario)
VALUES (1, 'Ana García', 'Ventas', 50000);

-- Insertar múltiples filas
INSERT INTO empleados VALUES 
(2, 'Luis Pérez', 'Marketing', 55000),
(3, 'Marta Sánchez', 'Ventas', 52000);

3. UPDATE - Modificar Datos

-- Actualizar un registro
UPDATE empleados
SET salario = 53000
WHERE id = 1;

-- Actualizar múltiples campos
UPDATE empleados
SET departamento = 'Desarrollo', salario = 63000
WHERE id = 4;

⚠️ Regla de Oro: SIEMPRE usa WHERE en UPDATE y DELETE. Sin ella, modificarás/eliminarás TODAS las filas.

4. DELETE - Eliminar Datos

-- Eliminar un registro
DELETE FROM empleados WHERE id = 2;

-- Eliminar varios con condición
DELETE FROM empleados WHERE departamento = 'Marketing';

5. SELECT - Consultar Datos

-- Ver todas las columnas
SELECT * FROM empleados;

-- Ver columnas específicas
SELECT nombre, departamento, salario FROM empleados;

-- Valores únicos (sin duplicados)
SELECT DISTINCT departamento FROM empleados;

-- Filtrar con WHERE
SELECT * FROM empleados WHERE departamento = 'Ventas';

-- Operadores de comparación
SELECT * FROM empleados WHERE salario > 55000;

📝 Resumen: DML permite manipular los datos de la tabla. INSERT añade filas, UPDATE modifica valores, DELETE elimina filas, y SELECT recupera información. El WHERE es crucial para especificar qué filas afectan las operaciones.

EJERCICIO RESUELTO

Enunciado:

Dada la tabla EMPLEADOS, realizar: a) Insertar nuevo empleado, b) Aumentar 10% a empleados de Ventas, c) Mostrar nombre y salario de quienes ganan > 55000 o son de TI

Solución:
a) INSERT INTO empleados (nombre, departamento, salario) VALUES ('Juan', 'Contabilidad', 45000);
b) UPDATE empleados SET salario = salario * 1.10 WHERE departamento = 'Ventas';
c) SELECT nombre, salario FROM empleados WHERE salario > 55000 OR departamento = 'TI';
